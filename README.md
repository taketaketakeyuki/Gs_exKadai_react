# Gs_exKadai_react

## 課題内容（どんな作品か） 
- ジャンケンアプリと同じ形ですが、、、APIを使って、鍛えたい箇所を送るとトレーニングがランダムで返ってくる仕様を作りました。 例えば下記をコンソールにぶち込むとあたいが返ってきます。
- curl -X POST -H "Content-Type: application/json" -d "{\"mymuscle\":\"胸\"}" localhost:3001/training
